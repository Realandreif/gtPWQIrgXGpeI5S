PHOTOGRAMMETRY - STATUE OF A WOMAN (2023)

This is a personal project where I generated a photogrammetric model of a statue using terrestrial photography. My smartphone camera's burst shot mode allowed me to take around 700 photos while I circled the statue. After processing the images with Agisoft Metashape, I found it surprising how well my 3d model looked compared to the original, as all my photos were taken within minutes.

PHOTOGRAMMETRY - RURAL ONTARIO HOUSE (2023)

This is a photogrammetric model of my house that I created with the help of my recently acquired drone and Agisoft Metashape. Using aerial and terrestrial photography, I took 169 photos, which were sufficient to create a detailed model of the house on all sides. Using this model, as well as total station measurements, I can create a model with accurate dimensions which I can then use to generate orthophotos.
